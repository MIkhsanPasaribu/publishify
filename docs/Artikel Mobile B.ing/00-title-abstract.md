## _Abstract_

_The increasing prevalence of mobile device usage for business operations has created demand for mobile applications that extend the capabilities of web-based enterprise systems. This research presents the development of a mobile application for the Publishify book publishing management system utilizing the Flutter framework. The application is designed to provide mobile accessibility for manuscript management workflows, enabling authors, editors, administrators, and printing operators to perform essential publishing activities from their mobile devices regardless of location or time constraints. Flutter was selected as the development framework based on its cross-platform capabilities that enable deployment to both Android and iOS platforms from a single codebase, its hot reload feature that accelerates development iteration, and its native-like performance characteristics. The application architecture implements a layered separation of concerns, distinguishing presentation, business logic, and data access layers, with integration to a RESTful API backend and WebSocket connectivity for real-time notification delivery. Key features implemented include manuscript status tracking with visual progress indicators, editorial review queue management with feedback submission capabilities, real-time push notifications, and role-based dashboard analytics. Performance evaluation using Google PageSpeed Insights demonstrated a score of 88 out of 100 for mobile mode, with Total Blocking Time measured at 10 milliseconds and Cumulative Layout Shift at 0. Functional testing achieved a 100 percent success rate across all 24 test cases covering authentication, manuscript management, review system, and notification modules. Compatibility testing across five Android devices with varying specifications confirmed consistent functionality and visual rendering. The application successfully extends the web-based Publishify system to mobile platforms, providing stakeholders with enhanced accessibility and flexibility in managing book publishing processes while maintaining feature parity with the web interface for core functionality._

_Keywords: book publishing, cross-platform development, Flutter, manuscript management, mobile application_

## PENGEMBANGAN APLIKASI MOBILE PUBLISHIFY UNTUK SISTEM MANAJEMEN PENERBITAN BUKU MENGGUNAKAN FRAMEWORK FLUTTER

### Abstrak

Meningkatnya prevalensi penggunaan perangkat mobile untuk operasi bisnis telah menciptakan permintaan akan aplikasi mobile yang memperluas kemampuan sistem enterprise berbasis web. Penelitian ini menyajikan pengembangan aplikasi mobile untuk sistem manajemen penerbitan buku Publishify menggunakan framework Flutter. Aplikasi dirancang untuk menyediakan aksesibilitas mobile bagi alur kerja manajemen naskah, memungkinkan penulis, editor, administrator, dan operator percetakan melakukan aktivitas penerbitan esensial dari perangkat mobile mereka tanpa batasan lokasi atau waktu. Flutter dipilih sebagai framework pengembangan berdasarkan kemampuan lintas platformnya yang memungkinkan deployment ke platform Android dan iOS dari satu codebase, fitur hot reload yang mempercepat iterasi pengembangan, dan karakteristik performa yang menyerupai aplikasi native. Arsitektur aplikasi mengimplementasikan pemisahan lapisan yang membedakan lapisan presentasi, logika bisnis, dan akses data, dengan integrasi ke backend API RESTful dan konektivitas WebSocket untuk pengiriman notifikasi real-time. Fitur-fitur utama yang diimplementasikan meliputi pelacakan status naskah dengan indikator progres visual, manajemen antrian review editorial dengan kemampuan pengiriman feedback, notifikasi push real-time, dan analitik dasbor berbasis peran. Evaluasi performa menggunakan Google PageSpeed Insights menunjukkan skor 88 dari 100 untuk mode mobile, dengan Total Blocking Time terukur pada 10 milidetik dan Cumulative Layout Shift pada 0. Pengujian fungsional mencapai tingkat keberhasilan 100 persen pada seluruh 24 kasus uji yang mencakup modul autentikasi, manajemen naskah, sistem review, dan notifikasi. Pengujian kompatibilitas pada lima perangkat Android dengan spesifikasi berbeda mengkonfirmasi konsistensi fungsionalitas dan rendering visual. Aplikasi berhasil memperluas sistem Publishify berbasis web ke platform mobile, memberikan pemangku kepentingan aksesibilitas dan fleksibilitas yang lebih baik dalam mengelola proses penerbitan buku sambil mempertahankan paritas fitur dengan antarmuka web untuk fungsionalitas inti.

**Kata kunci:** aplikasi mobile, Flutter, lintas platform, manajemen naskah, penerbitan buku
