## Catatan Referensi Berkas

Untuk mengakses kode sumber dan dokumentasi yang direferensikan dalam lampiran, silakan merujuk pada struktur direktori proyek berikut:

```
publishify/
├── backend/                    → Lampiran A.1, A.4, C.1
│   ├── src/                   → Kode sumber backend
│   ├── prisma/                → Skema basis data
│   └── swagger-endpoints.json → Dokumentasi API
├── frontend/                   → Lampiran A.2
│   └── app/                   → Kode sumber frontend
├── mobile/                     → Lampiran A.3
│   └── lib/                   → Kode sumber aplikasi seluler
├── docs/                       → Lampiran E.1
│   └── deployment-guide.md    → Panduan penyebaran
└── .github/                    → Lampiran F.1
    └── copilot-instructions.md → Petunjuk pengembangan
```
