### _Abstract_

_This research presents the development of Publishify, a comprehensive web-based book publishing information system designed to digitalize and streamline the manuscript management workflow from submission to publication. The system was developed using modern web technologies including Next.js 16 for the frontend and NestJS 10 for the backend, integrated with PostgreSQL database managed through Prisma ORM. The development methodology employed was the waterfall model with iterative refinement. The system implements role-based access control supporting four user roles: author (penulis), editor, admin, and printing operator (percetakan). Key features include manuscript submission and tracking, editorial review workflow with feedback system, publishing package management, real-time notifications via WebSocket, and comprehensive dashboard analytics. Performance testing using Google PageSpeed Insights demonstrated excellent results with a performance score of 98/100 for desktop, First Contentful Paint of 0.4 seconds, and Cumulative Layout Shift of 0. Functional testing achieved 100% success rate across all 28 test cases. The system successfully addresses the inefficiencies of traditional publishing workflows by providing a centralized digital platform for all stakeholders in the publishing process._

### Abstrak

Penelitian ini menyajikan pengembangan Publishify, sebuah sistem informasi penerbitan buku berbasis web yang komprehensif yang dirancang untuk mendigitalisasi dan menyederhanakan alur kerja manajemen naskah dari pengajuan hingga penerbitan. Sistem dikembangkan menggunakan teknologi web modern termasuk Next.js 16 untuk antarmuka pengguna dan NestJS 10 untuk layanan backend, terintegrasi dengan basis data PostgreSQL yang dikelola melalui Prisma ORM. Metodologi pengembangan yang digunakan adalah model waterfall dengan penyempurnaan iteratif. Sistem mengimplementasikan kontrol akses berbasis peran yang mendukung empat peran pengguna: penulis, editor, admin, dan operator percetakan. Fitur-fitur utama meliputi pengajuan dan pelacakan naskah, alur kerja review editorial dengan sistem umpan balik, manajemen paket penerbitan, notifikasi waktu nyata melalui WebSocket, dan analitik dasbor yang komprehensif. Pengujian performa menggunakan Google PageSpeed Insights menunjukkan hasil yang sangat baik dengan skor performa 98/100 untuk desktop, First Contentful Paint 0,4 detik, dan Cumulative Layout Shift 0. Pengujian fungsional mencapai tingkat keberhasilan 100% di seluruh 28 kasus uji. Sistem berhasil mengatasi ketidakefisienan alur kerja penerbitan tradisional dengan menyediakan platform digital terpusat untuk semua pemangku kepentingan dalam proses penerbitan.
