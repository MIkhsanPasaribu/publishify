### _Abstract_

_This research presents the development of a mobile application for the Publishify book publishing management system using the Flutter framework. The application is designed to provide mobile accessibility for the manuscript management workflow, enabling authors, editors, and administrators to manage publishing activities anytime and anywhere through their mobile devices. Flutter was selected as the development framework due to its cross-platform capabilities, hot reload feature, and native-like performance. The application implements a layered architecture separating presentation, business logic, and data layers, integrated with a RESTful API backend and WebSocket for real-time notifications. Key features include manuscript status tracking, editorial review queue management, real-time notifications, and role-based dashboard analytics. Performance testing using Google PageSpeed Insights showed a score of 88/100 for mobile mode with Total Blocking Time of 10 milliseconds and Cumulative Layout Shift of 0. Functional testing achieved 100% success rate across all 24 test cases covering authentication, manuscript management, review system, and notification modules. The application successfully extends the web-based Publishify system to mobile platforms, providing stakeholders with enhanced accessibility and flexibility in managing the book publishing process._

### Abstrak

Penelitian ini menyajikan pengembangan aplikasi mobile untuk sistem manajemen penerbitan buku Publishify menggunakan framework Flutter. Aplikasi dirancang untuk menyediakan aksesibilitas mobile bagi alur kerja manajemen naskah, memungkinkan penulis, editor, dan administrator mengelola aktivitas penerbitan kapan saja dan di mana saja melalui perangkat mobile mereka. Flutter dipilih sebagai framework pengembangan karena kemampuan lintas platformnya, fitur hot reload, dan performa yang menyerupai aplikasi native. Aplikasi mengimplementasikan arsitektur berlapis yang memisahkan lapisan presentasi, logika bisnis, dan data, terintegrasi dengan backend API RESTful dan WebSocket untuk notifikasi real-time. Fitur-fitur utama meliputi pelacakan status naskah, manajemen antrian review editorial, notifikasi real-time, dan analitik dasbor berbasis peran. Pengujian performa menggunakan Google PageSpeed Insights menunjukkan skor 88/100 untuk mode mobile dengan Total Blocking Time 10 milidetik dan Cumulative Layout Shift 0. Pengujian fungsional mencapai tingkat keberhasilan 100% pada seluruh 24 kasus uji yang mencakup modul autentikasi, manajemen naskah, sistem review, dan notifikasi. Aplikasi berhasil memperluas sistem Publishify berbasis web ke platform mobile, memberikan pemangku kepentingan aksesibilitas dan fleksibilitas yang lebih baik dalam mengelola proses penerbitan buku.
